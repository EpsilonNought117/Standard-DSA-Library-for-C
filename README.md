This is a simple library for data structures and algorithms, implemented in C.

1. Vector (Dynamic Array)
2. Singly Linked List
3. Queue
4. Stack 
5. Deque

Sorting Algorithms:

1. Bubble Sort
2. Selection Sort
3. Insertion Sort
4. Binary Insertion Sort
5. Merge Sort
6. Heap Sort
7. Quick Sort
